Users Routes

Path HTTP methods Description
/users GET Retrieve all users
/users/:id GET Retrieve a specific user by ID
/users/register Post To add a specific user
/user/update/:id Put/Patch To update a specific user
/user/delete/:id Delete To delete a specific user
